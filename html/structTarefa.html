Descrição detalhada

Estrutura de lista para as tarefas do grafo contendo o Id da tarefa(id_tarefa), o nome da tarefa (nome_tarefa), se a tarefa já foi executada ou não (tarefa_executada), a duração da tarefa (duracao_tarefa), o periodo minimo em que uma tarefa pode iniciar (inicio_min_tarefa), o número de pre requisitos dessa tarefa (n_prerequisitos), um ponteiro para apontar para o próximo elemento (*prox) e um ponteiro para apontar para a lista de pré requisitos dessa tarefa (*prerequisitos_tarefa).
